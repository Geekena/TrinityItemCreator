--------------
## Shortcut keys:
* [CTRL+S] Export as *.SQL
* [CTRL+D] Save to Clipboard
* [CTRL+F] Search DisplayID
* [ALT+D] Shows Default Templates Menu
* [ALT+C] Shows Custom Templates List
* [ALT+S] Save Custom Template
* [ALT+G] Shows Generate Stats Menu
